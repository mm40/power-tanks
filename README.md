<h1 align="center"> Power Tanks </h2>


## Keyboard Shortcuts

<center>

| **Shortcut**      | **Action**                                 |
|-------------------|--------------------------------------------|
| W, S, A, D        | Tank Movement                              |
| Space             | Fire                                       |
| Left/Right Arrows | Rotate Turret                              |
| R                 | Switch Ammo                                |
| Shift             | Fire/Activate Pizza round (when collected) |
| Ctrl              | Deploy shield (when collected)             |
| Escape            | Toggle pause                               |
|                   |                                            |

</center>

## Installation
### Windows
1. Download `PowerTanks.swf` and `FlashPlayer.exe`
2. Drag and drop the `.swf` file in `FlashPlayer.exe`
### Linux
Flash Player for Linux doesn't work too well, so using a Windows virtual machine for steps 1. and 2. is recommended.
